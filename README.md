# anipy-cli-launcher
Simple cmd script to use with anipy-cli
<img width="664" height="408" alt="image" src="https://github.com/user-attachments/assets/08b85e9b-fb5a-4b5b-9ba0-0a6ab9873d86" />

## Installation
1. Install [anipy-cli](https://sdaqo.dev/anipy-cli/getting-started-cli/#options-to-install)
2. Clone this repo or manually download anipy-cli-launcher.bat
3. Launch anipy-cli-launcher.bat
